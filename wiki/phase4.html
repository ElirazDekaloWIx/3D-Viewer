מה זה ולמה צריך את זה

צללים הם מה שמעגן אובייקט בסצנה. בלי צללים, מודל נראה כאילו הוא "מרחף" באוויר. צל טוב נותן תחושה של משקל, עומק, ומרחב.

למה צללים "רגילים" לא מספיקים?

צללים בסיסיים במנועי תלת-ממד הם חדים ושטוחים — כמו צל בצהריים חמים. אבל במציאות, צללים הם רכים — חדים ליד הבסיס ומטושטשים ככל שמתרחקים:

🌉 הפנומברה

שימו לב לצל של האצבע שלכם על שולחן: ממש ליד האצבע הצל חד, אבל ככל שמרחיקים את היד מהשולחן — הצל נעשה מטושטש יותר ויותר. האזור המטושטש נקרא פנומברה (Penumbra).

PCSS (Percentage-Closer Soft Shadows) מדמה בדיוק את התופעה הזו — צל שמשתנה ברוחבו בהתאם למרחק מהאובייקט שמטיל אותו.

🌏 צל מגע (Contact Shadow)

צל עדין מאוד שנוצר בדיוק במקום שבו האובייקט נוגע ברצפה. גם אם כל הצללים האחרים כבויים — צל מגע לבדו יכול לעגן את האובייקט בצורה משכנעת. זה מה ש-Sketchfab ו-Stager משתמשים.

אלגוריתם PCSS — שלושה שלבים

שלב 1: חיפוש חוסמים (Blocker Search)

דגימת מפת הצל (shadow map) באזור רחב סביב הפיקסל הנוכחי. המטרה: למצוא מהו העומק הממוצע של האובייקטים שחוסמים אור. משתמש ב-Poisson Disk לדגימה אחידה.

שלב 2: חישוב רוחב הפנומברה

penumbraWidth = (receiverDepth - blockerDepth) * lightSize / blockerDepth

ככל שהמרחק בין האובייקט החוסם למשטח המקבל גדול יותר — הפנומברה רחבה יותר (צל מטושטש יותר).

שלב 3: סינון PCF משתנה

דגימת מפת הצל שוב, הפעם עם רדיוס Poisson Disk שמותאם לרוחב הפנומברה שחושב. קרוב לאובייקט = דגימה צפופה (חד), רחוק = דגימה רחבה (מטושטש).

הגדרות מפת צל

// shadow-system.ts
function configureShadows(light, sceneBounds) {
  light.castShadow = true;
  light.shadow.mapSize.set(2048, 2048);
  light.shadow.type = VSMShadowMap; // or PCFSoftShadowMap

  // Auto-fit shadow camera to scene
  const d = sceneBounds.max.distanceTo(sceneBounds.min) * 0.6;
  light.shadow.camera.left = -d;
  light.shadow.camera.right = d;
  light.shadow.camera.top = d;
  light.shadow.camera.bottom = -d;
  light.shadow.camera.near = 0.1;
  light.shadow.camera.far = d * 4;

  // Bias to prevent shadow acne
  light.shadow.bias = -0.0005;
  light.shadow.normalBias = 0.02;
}

🌞 Ray-Traced Shadows ב-WebGPU

עם WebGPU Compute Shaders אפשר לממש צללים מבוססי מעקב קרניים במקום מפות צל. היתרון: אין shadow acne, אין peter panning, ואין מגבלת רזולוציה. החיסרון: דורש BVH (מבנה האצה) ויותר חישוב.

אפשרי כשדרה לפרויקט ב-Phase 8 (Path Tracing).

👁 Moment Shadow Maps (MSM)

שיטה מתקדמת שמאחסנת את "המומנטים" של התפלגות העומק במפת הצל. פותרת את בעיית light bleeding של VSM ונותנת צללים חלקים יותר. דורשת 4 ערכים לכל פיקסל (במקום 1), אבל התוצאות מצוינות.
