הסבר פשוט
מה זה ולמה צריך את זה

דמיינו שאתם בונים מסגרת של תמונה. לפני שתוכלו לצייר משהו — אתם צריכים את הקנבס, את המעמד, ואת הצבעים. ה"יסודות" הם בדיוק זה: המסגרת הבסיסית שמאפשרת להציג דברים על המסך.

שלושת הרכיבים:

🎨 הרנדרר (Renderer)

המנוע שמצייר פיקסלים על המסך. כמו מדפסת — הוא מקבל נתונים תלת-ממדיים ו"מדפיס" אותם כתמונה דו-ממדית. יש לנו שני סוגי "מדפסות":

  • WebGPU — הדור החדש, מהיר פי 2-10, תומך בחישובים מתקדמים. כמו מדפסת לייזר.
  • WebGL2 — הוותיק והמוכר, עובד בכל דפדפן. כמו מדפסת הזרקת דיו — אמינה ונגישה.

הויואר שלנו ינסה קודם את WebGPU, ואם הדפדפן לא תומך — ייפול בחן ל-WebGL2.

📦 טוען המודלים (Asset Loader)

מודל תלת-ממדי הוא קובץ שמכיל גיאומטריה (צורות), טקסטורות (תמונות שמודבקות על הצורות), וחומרים (איך הדבר נראה — מבריק? מט? שקוף?).

הקובץ הגיע דחוס כדי שייטען מהר. הטוען שלנו יודע לפרק את הדחיסה ברקע (ב-WebWorker — חוט עבודה נפרד שלא מאט את הממשק), כך שהמשתמש לא מרגיש שהדפדפן "נתקע".

🎥 בקרת מצלמה (Camera Controls)

כמו מצלמה אמיתית שמסתובבת סביב האובייקט. המשתמש יכול:

  • לסובב (גרירה עם העכבר) — ראה את המודל מכל זווית
  • לזום (גלגלת העכבר) — התקרב/התרחק
  • להזיז (Pan) — הזז את המרכז

המצלמה גם יודעת להתאים את עצמה אוטומטית לגודל המודל — מודל קטן? זום פנימה. מודל ענק? זום החוצה.

אסטרטגיית Fallback של הרנדרר

הגישה מבוססת על dynamic import — ניסיון ייבוא של WebGPU ונפילה חיננית:

// renderer.ts - Renderer Factory
export async function createRenderer(canvas: HTMLCanvasElement) {
  try {
    // Try WebGPU first
    const { WebGPURenderer } = await import('three/webgpu');
    const renderer = new WebGPURenderer({ canvas, antialias: true });
    await renderer.init();  // WebGPU requires async init!

    renderer.toneMapping = ACESFilmicToneMapping;
    renderer.outputColorSpace = SRGBColorSpace;
    renderer.setPixelRatio(Math.min(devicePixelRatio, 2.0));

    return { renderer, isWebGPU: true };
  } catch {
    // Fallback to WebGL2
    const { WebGLRenderer } = await import('three');
    const renderer = new WebGLRenderer({ canvas, antialias: true });

    renderer.toneMapping = ACESFilmicToneMapping;
    renderer.outputColorSpace = SRGBColorSpace;
    renderer.setPixelRatio(Math.min(devicePixelRatio, 2.0));

    return { renderer, isWebGPU: false };
  }
}
💡 הערה חשובה

WebGPURenderer דורש await renderer.init() לפני כל פעולת רנדור. זו קריאה אסינכרונית — בניגוד ל-WebGLRenderer שעובד מיידית.

הרכבת מערך הטעינה

// asset-loader.ts - Loader Composition
import { GLTFLoader } from 'three/addons/loaders/GLTFLoader.js';
import { DRACOLoader } from 'three/addons/loaders/DRACOLoader.js';
import { KTX2Loader } from 'three/addons/loaders/KTX2Loader.js';
import { MeshoptDecoder } from 'three/addons/libs/meshopt_decoder.module.js';

export function createAssetLoader(renderer: WebGPURenderer | WebGLRenderer) {
  // Draco - geometry compression (90-95% size reduction)
  const dracoLoader = new DRACOLoader();
  dracoLoader.setDecoderPath('/draco/');  // WASM decoders

  // KTX2 - texture compression (82% size reduction)
  const ktx2Loader = new KTX2Loader();
  ktx2Loader.setTranscoderPath('/basis/');
  ktx2Loader.detectSupport(renderer);

  // GLTF - the main loader
  const gltfLoader = new GLTFLoader();
  gltfLoader.setDRACOLoader(dracoLoader);
  gltfLoader.setKTX2Loader(ktx2Loader);
  gltfLoader.setMeshoptDecoder(MeshoptDecoder);

  return gltfLoader;
}

התאמת מצלמה אוטומטית

// Auto-fit camera to model bounding box
function fitCameraToModel(camera, controls, model) {
  const box = new Box3().setFromObject(model);
  const center = box.getCenter(new Vector3());
  const size = box.getSize(new Vector3());

  const maxDim = Math.max(size.x, size.y, size.z);
  const fov = camera.fov * (Math.PI / 180);
  const distance = maxDim / (2 * Math.tan(fov / 2));

  camera.position.set(
    center.x + distance * 0.5,
    center.y + distance * 0.3,
    center.z + distance
  );
  controls.target.copy(center);
  controls.update();
}

⚡ Meshopt vs Draco — השוואת דחיסת גיאומטריה

שני הפורמטים דוחסים גיאומטריה, אבל עם גישות שונות:

  • Draco: דחיסה אגרסיבית (90-95%), אבל פענוח איטי יותר. מצוין לקבצים גדולים שעוברים ברשת.
  • Meshopt: דחיסה מתונה יותר (~70%), אבל פענוח מהיר מאוד ומאפשר streaming — הצגת חלקים של המודל לפני שהכל נטען.

אפשר לתמוך בשניהם ולבחור לפי מקרה השימוש.
