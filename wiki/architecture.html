ארכיטקטורה

מבנה המערכת, תרשים Pipeline, וזרימת נתונים

עקרונות תכנון

⛏ מודולריות

כל מערכת (חומרים, תאורה, צללים, אפקטים, ביצועים) היא מודול עצמאי שמתקשר עם האחרים רק דרך Event Bus מטופס. אפשר לפתח, לבדוק, ולהחליף כל מודול בנפרד.

🔄 Progressive Enhancement

WebGPU ראשי עם נפילה חיננית ל-WebGL2. הדגל isWebGPU נבדק בדיוק בשני מקומות: מערכת החומרים (בחירה בין Node Materials ל-Standard Materials) ושרשרת האפקטים (בחירה בין TSL PostProcessing ל-EffectComposer).

⚡ Adaptive Quality

מכונת מצבים שמודדת FPS בזמן-אמת ומתאימה את רמת האיכות אוטומטית. שיפור ושיפוט (upgrade/degrade) עם hysteresis למניעת "קפיצות" מעצבנות.

תרשים Rendering Pipeline

📦
Input: Model File (.glb / .gltf)
↓
⚙
Decompress: Draco (geometry, WebWorker) + KTX2 (textures, WebWorker)
↓
🔧
Assemble: GLTFLoader → Scene Graph with meshes, materials, animations
↓
🎨
Upgrade Materials: KHR extensions → MeshPhysicalMaterial properties
↓
☀
Lighting: HDR environment (PMREM IBL) + 3-point light rig
↓
🌉
Shadows: PCSS shadow maps (variable penumbra)
↓
📸
Rasterize: WebGPURenderer / WebGLRenderer → HDR framebuffer
↓
✨
Post-Process: SSAO → SSR → Bloom → ACES Tone Map → AA → Vignette
↓
🖥
Output: Final pixel → sRGB display
🔄 מצב Path Tracing (אופציונלי)

כשמופעל, מחליף את שלבי Rasterize ו-Post-Process ב-Compute Shader שמבצע מעקב קרניים מלא + denoising. הפלט מוזן ישירות ל-tone mapping ו-AA.

זרימת תקשורת בין מודולים

Event Bus - Typed Messages

AssetLoader ──'model:loaded'──▶ SceneManager (add to scene)
                                    ├──▶ MaterialSystem (traverse & upgrade)
                                    ├──▶ ShadowSystem (configure shadow maps)
                                    ├──▶ AnimationManager (register clips)
                                    └──▶ LightRig (fit to model bounds)

FrameBudget ──'quality:changed'──▶ ShadowSystem (adjust resolution)
                                     ├──▶ PostProcessing (toggle effects)
                                     ├──▶ TextureManager (cap resolution)
                                     └──▶ LODManager (adjust bias)

Environment ──'env:loaded'──▶ Scene (set environment map)

User API    ──'settings:*'──▶ All modules (reconfigure)
