🧴 אנימציית של�� (Skeletal Animation)

כמו בובת מריונטה — יש "שלד" (מערכת עצמות) בתוך המודל, והעור (הגיאומטריה) נע בהתאם לעצמות. ככה דמויות הולכות, מניפות ידיים, ומסתובבות. כל תנועה מוגדרת כ"קליפ" (Animation Clip) שאפשר להפעיל, לעצור, ולערבב.

💫 אנימציית מורף (Morph Targets)

במקום שלד, הצורה עצמה "מתעוותת" בין מצבים שונים. חשבו על פרצוף — מצב אחד חייכני, מצב אחד עצוב, מצב אחד מופתע. על ידי ערבוב (0% עד 100%) בין המצבים, אפשר ליצור כל הבעה. נפוץ גם ל-blend shapes בתעשיית הקולנוע.

Three.js Animation System

// animation-manager.ts
import { AnimationMixer, AnimationClip, LoopRepeat } from 'three';

export class AnimationManager {
  private mixer: AnimationMixer;
  private clips: Map<string, AnimationAction> = new Map();
  private activeAction: AnimationAction | null = null;

  init(model, gltfAnimations: AnimationClip[]) {
    this.mixer = new AnimationMixer(model);

    for (const clip of gltfAnimations) {
      const action = this.mixer.clipAction(clip);
      this.clips.set(clip.name, action);
    }
  }

  play(name: string, crossFade = 0.3) {
    const action = this.clips.get(name);
    if (!action) return;

    if (this.activeAction) {
      action.reset().crossFadeFrom(this.activeAction, crossFade, true);
    }
    action.play();
    this.activeAction = action;
  }

  update(deltaTime: number) {
    this.mixer?.update(deltaTime);
  }

  getClipNames(): string[] {
    return Array.from(this.clips.keys());
  }
}

🧴 GPU Skinning ב-Compute Shaders

אנימציית שלד סטנדרטית מחשבת תנוחות על ה-CPU או ב-vertex shader. עם WebGPU Compute Shaders אפשר לבצע את החישוב ב-compute pass נפרד — מאפשר הנפשה של עשרות אלפי דמויות במקביל (חשוב לסצנות עם המונים).

🎬 glTF Animation Pointer Extension

הרחבה חדשה (KHR_animation_pointer) שמאפשרת להנפיש כל תכונה של כל אובייקט בסצנה — לא רק טרנספורמציות ומורפים, אלא גם צבעים של חומרים, עוצמת אור, מיקום מצלמה, וכו'.
