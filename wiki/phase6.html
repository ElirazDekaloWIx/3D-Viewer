מה זה ולמה צריך את זה

אין טעם בויואר יפהפה שלוקח 30 שניות לטעון ורץ ב-15 פריימים לשנייה. Sketchfab מצליח להציג מודלים כבדים בצורה חלקה כי הוא עושה 8 מעברי אופטימיזציה על כל קובץ שמעלים, ומתאים את האיכות למכשיר אוטומטית.

📸 טעינה פרוגרסיבית — הצג משהו מיד

במקום לחכות שהכל ייטען — מציגים תוצאה חלקית מיד ומשפרים בהדרגה:

  • שלב 1 (מיידי): קווי מתאר של תיבה חוסמת — המשתמש רואה "משהו"
  • שלב 2 (1-2 שניות): הגיאומטריה עם חומרים בסיסיים — הצורה ניכרת
  • שלב 3 (2-5 שניות): טקסטורות מלאות באיכות עולה — התוצאה הסופית
⚙ איכות אדפטיבית — המכשיר קובע

כמו משחקי וידיאו שמתאימים גרפיקה לכרטיס המסך — הויואר מזהה את ביצועי המכשיר ומכוונן את עצמו אוטומטית. טלפון ישן? פחות אפקטים. מחשב גיימינג? הכל על מקסימ��ם.

📦 דחיסה — פחות נתונים, אותו מראה

דחיסת טקסטורות (KTX2 + Basis Universal): מקטינה קבצי תמונה ב-82% ומאפשרת ל-GPU לקרוא אותם ישירות בלי פענוח.

דחיסת גיאומטריה (Draco): מקטינה נתוני צורה ב-90-95%. במקום 60MB קובץ, רק 3-6MB.

טקסטורות אדפטיביות (כמו Sketchfab)

// texture-manager.ts - Multi-resolution textures
// Sketchfab generates 5 resolution tiers per texture:
// 256, 512, 1024, 2048, 4096
// And selects based on device + viewport size

function selectTextureResolution() {
  const gpu = navigator.gpu;               // WebGPU available?
  const memory = navigator.deviceMemory;    // RAM hint
  const mobile = /Mobi|Android/i.test(navigator.userAgent);

  if (mobile)                   return 512;
  if (!gpu || memory < 4)      return 1024;
  if (memory >= 8)             return 4096;
  return 2048;
}

⚡ GPU-Driven Rendering Pipeline

במקום שה-CPU יחליט מה לצייר — ה-GPU עצמו מבצע frustum culling ובחירת LOD ב-compute shader, ושולח פקודות ציור דרך indirect draw buffers. מאפשר רנדור של מיליוני אובייקטים ב-60fps.

דורש WebGPU. יש מדריך מפורט עם מימוש.

💻 Meshopt Streaming

בעוד Draco דוחס יותר, Meshopt מאפשר streaming — פענוח והצגה של חלקים מהמודל לפני שהקובץ כולו הגיע. אידאלי לטעינה פרוגרסיבית.

🚀 BatchedMesh ב-Three.js

BatchedMesh (חדש ב-r160+) מאפשר ציור של אלפי אובייקטים ב-draw call אחד, גם כשיש להם גיאומטריות וחומרים שונים. שיפור ביצועים דרמטי לסצנות מורכבות.
