מה זה ולמה צריך את זה

תאורה היא מה שעושה את ההבדל בין "מודל תלת-ממדי שנראה כמו משחק ישן" לבין "תמונה שנראית כמו צילום". בדיוק כמו בצילום אמיתי — התאורה היא הכל.

שני סוגי תאורה:

🌎 תאורת סביבה (IBL - Image Based Lighting)

דמיינו שאתם עוטפים את המודל בתמונה ענקית של סביבה — שמים, עצים, קירות. האור מהתמונה הזו "נופל" על המודל מכל כיוון, בדיוק כמו במציאות.

זו השיטה שגורמת לכדור מתכת לשקף את הסביבה, ולעור מט להיראות רך ומואר באופן טבעי. זה נקרא IBL כי התאורה באה מתמונה (Image).

☀ תאורה ישירה (Direct Lights)

כמו פנסים בסטודיו צילום. מוסיפים הדגשות, צללים, ומבנה:

  • אור מפתח (Key Light) — האור הראשי, חזק, יוצר צללים
  • אור מילוי (Fill Light) — רך, ממתן צללים חזקים מדי
  • אור שפה (Rim Light) — מאחור, יוצר קו אור מסביב לאובייקט שמפריד אותו מהרקע
🌅 מפת HDR

תמונת HDR (High Dynamic Range) היא תמונה שמכילה יותר מידע על בהירות מאשר תמונה רגילה. היא יכולה לייצג גם את השמש הבוהקת וגם את הצל העמוק — בדיוק כמו העין האנושית. בלי HDR, ההשתקפויות נראות שטוחות ומזויפות.

תהליך PMREM — ממפת HDR לתאורת סביבה

PMREM (Prefiltered Mipmap Radiance Environment Map) הוא תהליך שמכין את מפת הסביבה לשימוש בחומרים. הוא יוצר גרסאות מטושטשות ברמות שונות — חומר חלק (מבריק) ישתמש בגרסה החדה, חומר מחוספס ישתמש בגרסה המטושטשת. 

// environment.ts - HDR to IBL Pipeline
import { RGBELoader } from 'three/addons/loaders/RGBELoader.js';
import { PMREMGenerator } from 'three';

export async function loadEnvironment(renderer, scene, hdrPath) {
  const pmremGenerator = new PMREMGenerator(renderer);
  pmremGenerator.compileEquirectangularShader();

  const rgbeLoader = new RGBELoader();
  const hdrTexture = await rgbeLoader.loadAsync(hdrPath);

  // Generate prefiltered cubemap (256x256, GGX importance sampling)
  const envMap = pmremGenerator
    .fromEquirectangular(hdrTexture)
    .texture;

  // Apply to scene
  scene.environment = envMap;        // Used by PBR materials
  scene.background = envMap;         // Optional: visible background
  scene.backgroundBlurriness = 0.5; // Blur the background

  // Cleanup
  hdrTexture.dispose();
  pmremGenerator.dispose();

  return envMap;
}

סט תאורת סטודיו

// light-rig.ts - Studio Lighting Presets
export function createStudioLighting(scene) {
  // Key light - main directional, casts shadows
  const keyLight = new DirectionalLight(0xffffff, 1.5);
  keyLight.position.set(5, 8, 5);
  keyLight.castShadow = true;

  // Fill light - softer, opposite side
  const fillLight = new DirectionalLight(0xb4c6e0, 0.6);
  fillLight.position.set(-4, 3, 2);

  // Rim light - backlight for edge definition
  const rimLight = new DirectionalLight(0xffeedd, 0.8);
  rimLight.position.set(-2, 6, -6);

  scene.add(keyLight, fillLight, rimLight);
  return { keyLight, fillLight, rimLight };
}

מישור רצפה עם צל מגע

// ground-plane.ts - Shadow-catching ground
export function createGroundPlane(scene) {
  const geometry = new PlaneGeometry(30, 30);
  const material = new ShadowMaterial({
    opacity: 0.3,  // Semi-transparent shadow
  });

  const plane = new Mesh(geometry, material);
  plane.rotation.x = -Math.PI / 2;
  plane.receiveShadow = true;

  scene.add(plane);
  return plane;
}
💡 טיפ: חשיפה (Exposure)

renderer.toneMappingExposure שולט בכמות האור הכוללת. ערך של 1.0 הוא ברירת מחדל, 0.5 חשוך יותר, 2.0 בהיר יותר. זה שקול ל-EV (Exposure Value) בצילום.

🌈 Radiance Cascades — תאורה גלובלית בזמן-אמת

טכניקה חדשנית שפותחה על ידי Alexander Sannikov (מפתח בכיר ב-Grinding Gear Games, יוצרי Path of Exile). שיטה זו מאפשרת חישוב Global Illumination (GI) — כלומר, אור שמנתר ממשטחים אחרים — ללא רעש ובזמן אמת.

הרעיון: חלוקת חישוב התאורה לכמה "קסקדות" ברזולוציות שונות. קסקדה גבוהה מטפלת בקרניים קצרות (פרטים עדינים), קסקדה נמוכה בקרניים ארוכות (תאורה כללית).

יתרונות: נטול רעש (בניגוד ל-Path Tracing), מהיר, עובד גם על חומרה חלשה.

חסרונות: כרגע רוב המימושים הם 2D. מימוש 3D קיים אך ניסיוני.

🧠 Neural Radiance Caching

טכניקה שפותחה על ידי NVIDIA שמשתמשת ברשת נוירונית קטנה כדי "לזכור" תוצאות חישוב תאורה ולמנוע חישובים כבדים חוזרים. במקום לזרוק קרניים לכל פיקסל מחדש — הרשת חוזה את התאורה על סמך מיקום ונורמל.

AMD פיתחה גרסה מסחרית בשם FSR Radiance Caching שמנבאת התפשטות אור בסצנה מדגימות Path Tracing דלילות, ומחליפה הרבה קרניים משניות בחיזויים נלמדים.

רלוונטיות לנו: ב-Phase 8 (Path Tracing), שימוש ב-Radiance Cache יכול להאיץ את ההתכנסות משמעותית.

🌞 Area Lights ב-Three.js

Three.js תומך ב-RectAreaLight — אורות שטח מלבניים שמדמים חלונות, מסכי טלוויזיה, או לוחות תאורה. הם יוצרים השתקפויות רכות ותאורה יותר ריאליסטית מאורות נקודתיים. דורשים RectAreaLightUniformsLib.

שימושי במיוחד להצגת מוצרים ב"חלל סטודיו" — ניתן להוסיף כאפשרות לסטי התאורה.
