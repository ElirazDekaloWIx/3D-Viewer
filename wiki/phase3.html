מה זה ולמה צריך את זה

חומרים (Materials) הם מה ש"מלביש" את הצורה התלת-ממדית. הם קובעים איך משטח מגיב לאור — האם הוא מבריק? מט? שקוף? זוהר? משקף?

PBR (Physically Based Rendering) אומר שהחומרים מתנהגים לפי חוקי הפיזיקה של אור אמיתי. זה מה שגורם לתמונה להיראות "נכונה" — מתכת נראית כמו מתכת, עור כמו עור, זכוכית כמו זכוכית.

סוגי החומרים שלנו:

💎 בסיסי — מתכתי/מחוספס (Metallic-Roughness)

כל חומר מוגדר על ידי שני מספרים פשוטים:

  • מתכתיות (Metallic): 0 = לא מתכת (פלסטיק, עץ, עור), 1 = מתכת (ברזל, זהב, כרום)
  • חספוס (Roughness): 0 = חלק לחלוטין (מראה), 1 = מחוספס לגמרי (גיר)

זה הבסיס. עם רק שני פרמטרים אלה אפשר כבר ליצור מגוון רחב של חומרים.

☀ שכבת ציפוי (Clearcoat)

כמו שכבת לכה שקופה מעל צבע. חשבו על צבע רכב מבריק — יש את הצבע עצמו, ומעליו שכבה שקופה מבריקה. או ציפורניים עם לק מבריק.

🌧 שקיפות והעברת אור (Transmission)

זכוכית, מים, קרח, פלסטיק שקוף. האור עובר דרך החומר, ויכול להישבר בדרך (כמו קש שנראה שבור בכוס מים). פרמ��ר IOR קובע כמה האור נשבר.

🌞 פיזור תת-שטחי (Subsurface Scattering - SSS)

כשמכניסים יד מול פנס חזק — רואים אור אדמדם שמפזר דרך העור. זה SSS. קורה גם בשעווה, שיש, חלב, ועלים. האור נכנס לחומר, מנתר בפנים, ויוצא ממקום אחר.

🧶 ברק בד (Sheen)

הברק הרך שרואים על קטיפה, משי, או ספה מבד כשהאור נופל בזווית. זה שונה מברק מתכתי — הוא רך ומפוזר, ונראה בעיקר בקצוות.

🔄 אנייזוטרופיה (Anisotropy)

מתכת מוברשת (כמו מקרר נירוסטה) משקפת אור בצורה מוארכת בכיוון ההברשה, לא בצורה עגולה אחידה. גם שיער, סיבי עץ, ותקליטורים מציגים תופעה זו.

🌈 אירידסנס (Iridescence)

צבעי הקשת שמשתנים בהתאם לזווית הצפייה — כמו בועות סבון, כנפי פרפר, שמן על מים.

💡 הערה

GLTFLoader של Three.js מטפל ברוב ההרחבות אוטומטית בגרסאות אחרונות. מערכת החומרים שלנו מוסיפה יכולת עריכה בזמן-ריצה ושליטה עדינה יותר.

קוד דוגמה: יצירת חומר זכוכית

// Glass material with transmission + IOR
const glassMaterial = new MeshPhysicalMaterial({
  color: 0xffffff,
  metalness: 0,
  roughness: 0.05,
  transmission: 1.0,     // Fully transparent
  thickness: 0.5,        // Refraction depth
  ior: 1.5,              // Glass IOR
  attenuationDistance: 0.5,
  attenuationColor: new Color(0xddeeff), // Slight blue tint
});

קוד דוגמה: חומר צבע רכב (Clearcoat)

// Car paint with clearcoat
const carPaint = new MeshPhysicalMaterial({
  color: 0xcc0000,       // Deep red
  metalness: 0.9,
  roughness: 0.4,
  clearcoat: 1.0,        // Full clearcoat layer
  clearcoatRoughness: 0.1, // Very smooth coat
});

BRDF — מודל Cook-Torrance

Three.js משתמש במודל Cook-Torrance BRDF לחישוב האינטראקציה בין אור לחומר. המודל מורכב משלושה חלקים:

  • D — Normal Distribution (GGX): מתאר את התפלגות הפני-שטח הזעירים (microfacets). חומר חלק = רוב הפנים מיושרים.
  • G — Geometry (Schlick-GGX): מודל הצללה עצמית של הפני-שטח הזעירים. חומר מחוספס = יותר הסתרה.
  • F — Fresnel (Schlick): חישוב כמה אור מוחזר מול כמה נספג, בהתאם לזווית. במבט ישיר — פחות השתקפות. בזווית חדה — יותר (כמו אגם שנראה כמו מראה כשמסתכלים מרחוק).

📚 OpenPBR — סטנדרט החומרים הבא

OpenPBR הוא סטנדרט חדש של ASWF (Academy Software Foundation) שמאחד את מודלי החומרים של Adobe Standard Material, Autodesk Standard Surface, ו-Pixar's MaterialX. יש מימוש ייחוס קוד פתוח של Adobe.

זה כולל חישובים מתקדמים של subsurface scattering (מעבר אקראי בנפח — random walk), דיספרסיה (פיצול צבעים דרך זכוכית, כמו פריזמה), ושכבת fuzz/sheen.

רלוונטיות: ב-Phase 8 (Path Tracing) נוכל להשתמש ב-OpenPBR BSDF לחישוב חומרים מדויק יותר.

GitHub: Adobe OpenPBR BSDF Reference  |  OpenPBR Surface Spec

🪟 Enterprise PBR — Dassault Systèmes

Dassault (יוצרי SolidWorks) פיתחו Enterprise PBR Shading Model — מודל חומרים מקיף שמכסה תעשיית הרכב, אופנה, ומוצרי צריכה. כולל מפרט מפורט מאוד של כל תכונות החומר עם נוסחאות מתמטיות, שימושי מאוד כרפרנס.

Enterprise PBR User Guide

🔮 Multiscattering — שיפור דיוק ההשתקפויות

מודל Cook-Torrance הסטנדרטי מתעלם מקפיצות אור מרובות בתוך הפני-שטח הזעירים (microfacets). זה גורם לאובדן אנרגיה — חומרים מחוספסים נראים חשוכים מדי. פתרון: Multiscattering GGX שמפצה על האנרגיה האבודה.

Three.js כבר כולל קירוב של פיצוי אנרגיה, אבל מימוש מלא עם LUT (טבלת חיפוש מחושבת מראש) ייתן תוצאות מדויקות יותר.

🎨 Dispersion — שבירת צבעים

כשאור עובר דרך זכוכית או יהלום, כל צבע נשבר בזווית אחרת — זה מה שיוצר קשת בפריזמה ואת ה"אש" ביהלומים. Three.js הוסיף תמיכה ב-dispersion דרך MeshPhysicalMaterial.dispersion.

שווה לשלב: במיוחד להצגת תכשיטים, כלי זכוכית, וגבישים.
